# Basics of Java

This folder contains fundamental concepts of Java programming.

## Topics Covered

### 1. Variables (`Variable.java`)

- **Definition**: Containers for storing data values.
- **Types**: Java is statically typed, meaning you must declare the variable type (e.g., `int`, `String`, `double`).
- **Example**: `int age = 25;`

### 2. Input (`input.java`)

- **Scanner Class**: Used to get user input.
- **Package**: Requires `import java.util.Scanner;`.
- **Methods**:
  - `next()`: Reads a single word.
  - `nextLine()`: Reads a whole line.
  - `nextInt()`: Reads an integer.

### 3. Basic Operations (`Sum.java`)

- Performing arithmetic operations like addition.
- Understanding operators (`+`, `-`, `*`, `/`).

### 4. Control Flow

- **If-Else (`IfElse.java`)**:
  - Decision-making statements.
  - Executes a block of code if a condition is true.
- **Switch Statements (`Switch.java`)**:
  - Selects one of many code blocks to be executed.
  - Useful when checking a single variable against multiple values (cases).

## How to Run

1.  Compile: `javac FileName.java`
2.  Run: `java FileName`
